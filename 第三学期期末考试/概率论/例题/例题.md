### 第一模块：概率论基础（四大公式、独立性、条件概率）

**【知识点详解】**

- **四大公式：** 加法公式、减法公式、乘法公式、全概率公式/贝叶斯公式。
    
- **相互独立：** $P(AB) = P(A)P(B)$。这是简化计算的利器。
    

【典型例题：全概率与贝叶斯】

某工厂有 A、B 二台机器生产同种产品，A 产量占 $60\%$，次品率为 $5\%$；B 产量占 $40\%$，次品率为 $10\%$。

1. 求从产品中任取一件是次品的概率。
    
2. 若已知取到的是次品，求它是由 A 机器生产的概率。
    

**【详细解析】**

1. 全概率公式： 设 $H_1$ 为 A 生产，$H_2$ 为 B 生产，$C$ 为次品。
    
    $P(C) = P(H_1)P(C|H_1) + P(H_2)P(C|H_2) = 0.6 \times 0.05 + 0.4 \times 0.10 = 0.07$。
    
2. **贝叶斯公式：** $P(H_1|C) = \frac{P(H_1)P(C|H_1)}{P(C)} = \frac{0.6 \times 0.05}{0.07} = \frac{3}{7} \approx 0.428$。
    
    > **考试技巧：** 第一问找“分流原因”，第二问是“由果索因”。

### 第二模块：随机变量及其分布（一维与二维）

**【知识点详解】**

- **归一化：** $\int f(x) dx = 1$。
    
- **边缘密度：** 在二维中，求 $f_X(x)$ 需对 $y$ 积分，反之亦然。**注意积分限随区域变化。**
    

【典型例题：二维连续型边缘分布】

设 $(X, Y)$ 的联合概率密度为 $f(x, y) = \begin{cases} 2, & 0 < x < y < 1 \\ 0, & \text{其他} \end{cases}$。求 $X$ 的边缘概率密度 $f_X(x)$。

**【详细解析】**

- **步骤 1：画图。** 区域 $D$ 是由 $y=x, y=1, x=0$ 围成的三角形。
    
- 步骤 2：积分。 求 $f_X(x)$，则固定 $x$，让 $y$ 在区域内滑动。
    
    当 $0 < x < 1$ 时，$y$ 的取值范围是从直线 $y=x$ 到 $y=1$。
    
    $f_X(x) = \int_{x}^{1} 2 dy = 2(1-x), \quad 0 < x < 1$。
    
    > **计信院考点：** 边缘分布的积分限确定是该章最容易扣分的地方。

### 第三模块：数据特征与极限定理（期望、方差、CLT）

**【知识点详解】**

- **期望与方差：** 掌握 $E(aX+b) = aE(X)+b$ 和 $D(aX+b) = a^2D(X)$。
    
- **切比雪夫不等式：** 用于估计概率上界 $P\{|X-E(X)| \geq \epsilon\} \leq \frac{D(X)}{\epsilon^2}$。
    

【典型例题：协方差与方差性质】

已知 $D(X)=4, D(Y)=9, \rho_{XY}=0.5$，求 $D(2X - Y)$。

**【详细解析】**

- **公式：** $D(aX + bY) = a^2D(X) + b^2D(Y) + 2ab \cdot Cov(X,Y)$。
    
- **计算：**
    
    1. $Cov(X,Y) = \rho_{XY} \cdot \sqrt{D(X)} \cdot \sqrt{D(Y)} = 0.5 \times 2 \times 3 = 3$。
        
    2. $D(2X-Y) = 2^2D(X) + (-1)^2D(Y) + 2(2)(-1)Cov(X,Y)$
        
    3. $D(2X-Y) = 4(4) + 1(9) - 4(3) = 16 + 9 - 12 = 13$。

### 第四模块：数理统计（参数估计、假设检验）

**【知识点详解】**

- **极大似然估计 (MLE)：** 核心是“取对数、求导、令为0”。
    
- **无偏性：** $E(\hat{\theta}) = \theta$ 则为无偏。样本均值 $\bar{X}$ 永远是 $E(X)$ 的无偏估计。
    

【典型例题：极大似然估计】

设总体 $X \sim E(\lambda)$，即密度为 $f(x) = \lambda e^{-\lambda x} (x>0)$。求 $\lambda$ 的极大似然估计量。

**【详细解析】**

1. **写出似然函数：** $L(\lambda) = \prod_{i=1}^n \lambda e^{-\lambda x_i} = \lambda^n e^{-\lambda \sum x_i}$。
    
2. **取对数：** $\ln L(\lambda) = n \ln \lambda - \lambda \sum x_i$。
    
3. **求导并令其为0：** $\frac{d \ln L}{d \lambda} = \frac{n}{\lambda} - \sum x_i = 0$。
    
4. **解得：** $\hat{\lambda} = \frac{n}{\sum x_i} = \frac{1}{\bar{X}}$。
    
    > **结论：** 指数分布参数的 MLE 就是样本均值的倒数。

### 📝 考前冲刺三句话：

1. **分布函数 $F(x)$ 必连续：** 在连续型随机变量题目中，如果要求分段函数的常数，利用 $F(x)$ 在分段点处的连续性极快。
    
2. **统计量不含未知参数：** 填空题如果问某个表达式是否是统计量，看里面有没有出现 $\mu$ 或 $\sigma$。
    
3. **假设检验看 $P$ 值：** $P$ 值小于 $\alpha$（显著性水平），拒绝原假设 $H_0$。
