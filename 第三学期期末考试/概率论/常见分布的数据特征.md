### 一、 核心考点与公式 (必须背诵)

这一章主要围绕四个概念：**期望 (Expectation)**、**方差 (Variance)**、**协方差 (Covariance)** 和 **相关系数 (Correlation)**。

#### 1. 数学期望 $E(X)$

- **离散型：** $E(X) = \sum_{k} x_k p_k$ (数值 $\times$ 概率，然后求和)
    
- **连续型：** $E(X) = \int_{-\infty}^{+\infty} x f(x) dx$ (数值 $\times$ 密度函数，然后积分)
    
- **重要性质（解题神器）：**
    
    - **线性性质：** $E(aX + b) = aE(X) + b$
        
    - **和的期望：** $E(X \pm Y) = E(X) \pm E(Y)$ (无论是否独立都成立)
        
    - **积的期望：** 仅当 $X, Y$ **独立**时，$E(XY) = E(X)E(Y)$
        

#### 2. 方差 $D(X)$ 或 $Var(X)$

- **定义式：** $D(X) = E\{[X - E(X)]^2\}$ (一般不用这个算)
    
- **计算公式（常用）：**
    
    $$D(X) = E(X^2) - [E(X)]^2$$
    
    - _口诀：方差等于“平方的期望”减去“期望的平方”。_
        
- **重要性质：**
    
    - $D(C) = 0$ (常数的方差为0)
        
    - $D(aX + b) = a^2 D(X)$ (**注意：系数a要平方，常数b扔掉**)
        
    - 若 $X, Y$ **独立**，则 $D(X \pm Y) = D(X) + D(Y)$ (**注意：减号也是变加号**)
        

#### 3. 协方差与相关系数 (二维随机变量)

- **协方差：** $Cov(X, Y) = E(XY) - E(X)E(Y)$
    
- **相关系数：** $\rho_{XY} = \frac{Cov(X, Y)}{\sqrt{D(X)}\sqrt{D(Y)}}$
    
- **一般加法公式：** $D(X \pm Y) = D(X) + D(Y) \pm 2Cov(X, Y)$

### 二、 常见分布的期望与方差 (背下来免计算)

期末考试中，如果题目给的是常见分布，**千万不要去积分**，直接套公式！

| **分布名称**  | **记号**             | **概率密度/分布律参数**                      | **期望 E(X)**         | **方差 D(X)**           |
| --------- | ------------------ | ----------------------------------- | ------------------- | --------------------- |
| **0-1分布** | $B(1, p)$          | $P(X=1)=p$                          | $p$                 | $p(1-p)$              |
| **二项分布**  | $B(n, p)$          | $n$次独立重复试验                          | $np$                | $np(1-p)$             |
| **泊松分布**  | $P(\lambda)$       | 强度 $\lambda$                        | $\lambda$           | $\lambda$             |
| **均匀分布**  | $U(a, b)$          | 区间 $[a, b]$                         | $\frac{a+b}{2}$     | $\frac{(b-a)^2}{12}$  |
| **指数分布**  | $E(\lambda)$       | $f(x)=\lambda e^{-\lambda x} (x>0)$ | $\frac{1}{\lambda}$ | $\frac{1}{\lambda^2}$ |
| **正态分布**  | $N(\mu, \sigma^2)$ | 参数 $\mu, \sigma$                    | $\mu$               | $\sigma^2$            |

_注意：指数分布有的教材写 $\theta$ 也就是 $\frac{1}{\lambda}$，请以你教材为准。通常记 $E(X)$ 为倒数。_

### 三、 经典例题与解题技巧![[Pasted image 20251224150913.png]]

#### 题型一：利用性质求数值 (最常见小题)

**题目：** 设随机变量 $X$ 的期望 $E(X)=3$，方差 $D(X)=2$。求 $E(X^2)$ 和 $D(2X-5)$。

**解答技巧：**

1. 求 $E(X^2)$ 利用方差倒推公式：$D(X) = E(X^2) - [E(X)]^2$
    
    $$E(X^2) = D(X) + [E(X)]^2 = 2 + 3^2 = 11$$
    
2. 求 $D(2X-5)$ 利用方差性质 $D(aX+b) = a^2D(X)$：
    
    $$D(2X-5) = 2^2 \cdot D(X) = 4 \times 2 = 8$$
    

#### 题型二：连续型变量求期望 (计算题)

**题目：** 设 $X$ 的概率密度为 $f(x) = \begin{cases} 2x, & 0 < x < 1 \\ 0, & \text{其他} \end{cases}$。求 $E(X)$ 和 $E(e^X)$。

解答技巧：

这是“随机变量函数的期望”，不需要先求 $Y=e^X$ 的密度函数，直接积分。

公式：$E[g(X)] = \int g(x)f(x)dx$

1. $$E(X) = \int_{0}^{1} x \cdot (2x) dx = \int_{0}^{1} 2x^2 dx = [\frac{2}{3}x^3]_0^1 = \frac{2}{3}$$
    
2. $$E(e^X) = \int_{0}^{1} e^x \cdot (2x) dx$$
    
    (此处使用分部积分法)
    
    $$= 2 [xe^x - e^x]_0^1 = 2(e - e - (0 - 1)) = 2$$
    

#### 题型三：二维分布求相关系数 (大题常客)

**题目：** 给出二维离散型分布律表格，或者给出二维密度函数 $f(x,y)$，求 $\rho_{XY}$。

**解题步骤 (标准套路)：**

1. **算边缘：** 如果是连续型，先积分求出边缘密度 $f_X(x)$ 和 $f_Y(y)$。
    
2. **算四个数字：**
    
    - $E(X)$
        
    - $E(Y)$
        
    - $D(X) = E(X^2) - E(X)^2$
        
    - $D(Y) = E(Y^2) - E(Y)^2$
        
3. **算混合期望：** $E(XY) = \iint xy f(x,y) dxdy$ (连续) 或 $\sum \sum x_i y_j p_{ij}$ (离散)。
    
4. **组合：**
    
    - $Cov(X,Y) = E(XY) - E(X)E(Y)$
        
    - $\rho = \frac{Cov}{\sqrt{D(X)D(Y)}}$

### 四、 避坑指南 (易错点)

1. **方差系数要平方：** 很多人算 $D(2X)$ 时候算出 $2D(X)$，错！应该是 $4D(X)$。
    
2. **独立 vs 不相关：**
    
    - 独立 $\Rightarrow$ 不相关 ($\rho=0$)。
        
    - 不相关 $\nRightarrow$ 独立 (除非是二维正态分布，此时二者等价)。
        
3. **切比雪夫不等式：** 虽然考得少，但如果考填空题，记得公式 $P\{|X-\mu| \ge \epsilon\} \le \frac{\sigma^2}{\epsilon^2}$。它主要用于已知期望方差，估算概率上界。
    
4. **0-1分布与二项分布：** 如果题目说“做一次实验”，是0-1分布；“做n次”，是二项分布。
