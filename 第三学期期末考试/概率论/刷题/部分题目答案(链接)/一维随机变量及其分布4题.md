这道题考查的是从**概率分布律（PMF）**推导**分布函数（CDF）**的方法。

对于离散型随机变量，$F(x) = P(X \le x)$。我们需要根据 $X$ 可能取的值（$0, 1, 2$），分段累加概率。

### 解题步骤

根据分布函数的定义，我们分区间讨论 $x$ 的取值：

1. 当 $x < 0$ 时：
    
    由于 $X$ 最小取 0，所以 $x$ 左侧没有概率点。
    
    $$F(x) = P(X \le x) = 0$$
    
2. 当 $0 \le x < 1$ 时：
    
    此时 $X \le x$ 范围内只包含 $X=0$ 这一个点。
    
    $$F(x) = P(X=0) = \frac{1}{2}$$
    
3. 当 $1 \le x < 2$ 时：
    
    此时 $X \le x$ 范围内包含了 $X=0$ 和 $X=1$ 两个点。
    
    $$F(x) = P(X=0) + P(X=1) = \frac{1}{2} + \frac{1}{3} = \frac{5}{6}$$
    
4. 当 $x \ge 2$ 时：
    
    此时 $X \le x$ 范围内包含了所有可能的点（$0, 1, 2$）。
    
    $$F(x) = P(X=0) + P(X=1) + P(X=2) = \frac{1}{2} + \frac{1}{3} + \frac{1}{6} = 1$$
    

---

### 最终答案

$X$ 的分布函数 $F(x)$ 为如下分段函数：

$$F(x) = \begin{cases} 0, & x < 0 \\ \frac{1}{2}, & 0 \le x < 1 \\ \frac{5}{6}, & 1 \le x < 2 \\ 1, & x \ge 2 \end{cases}$$

### 💡 技巧点拨

- **阶梯状图形**：离散型随机变量的分布函数图形一定是“阶梯状”的，在每一个概率点（如 $0, 1, 2$）处向上跳跃。
    
- **跳跃高度**：在 $x=1$ 处的跳跃高度（从 $1/2$ 跳到 $5/6$）正好等于 $P(X=1) = 1/3$。
    
- **右连续性**：注意分段区间中的等号通常写在左侧（如 $0 \le x < 1$），这体现了分布函数**右连续**的性质。
