这组题目将带你从“列表穷举法”进阶到“公式证明法”。我们要处理的是离散型随机变量最常见的几种变换：**最大值（Max）、最小值（Min）和差值（Difference）**。

---

## 核心知识点库

在开始解题前，请记住这两个“万能钥匙”：

1. **最大值 $M = \max(X, Y)$ 的逻辑**：
    
    - 事件 $\{M \le k\}$ 等价于 $\{X \le k \text{ 且 } Y \le k\}$。
        
    - 通俗点说：如果老大（最大那个）都没超过 $k$，那所有人肯定都没超过 $k$。
        
2. **最小值 $L = \min(X, Y)$ 的逻辑**：
    
    - 事件 $\{L > k\}$ 等价于 $\{X > k \text{ 且 } Y > k\}$。
        
    - 通俗点说：如果最差的那个人都比 $k$ 强，那所有人肯定都比 $k$ 强。
        
3. **独立变量的联合概率**：
    
    - 如果 $X, Y$ 独立，则 $P(X=i, Y=j) = P(X=i) \cdot P(Y=j)$。

## 11 & 12. 均匀分布的极值（矩阵列表法）

**题目背景**：$X, Y$ 在 $\{1, 2, 3\}$ 上服从均匀分布，意味着 $P(X=i) = 1/3$。因为独立，任意组合 $(i, j)$ 的概率都是 $1/3 \times 1/3 = 1/9$。

### 详细步骤：

我们可以画一个 $3 \times 3$ 的表格，填入对应的 $M$ 和 $L$ 值：

|**(X, Y)**|**Y=1**|**Y=2**|**Y=3**|
|---|---|---|---|
|**X=1**|(1,1)|(1,2)|(1,3)|
|**X=2**|(2,1)|(2,2)|(2,3)|
|**X=3**|(3,1)|(3,2)|(3,3)|

#### 第 11 题：$M = \max(X, Y)$ 分布律

- $M=1$：只有 (1,1) 这 **1** 种情况。$P(M=1) = 1/9$。
    
- $M=2$：有 (1,2), (2,1), (2,2) 这 **3** 种情况。$P(M=2) = 3/9 = 1/3$。
    
- $M=3$：有 (1,3), (2,3), (3,1), (3,2), (3,3) 这 **5** 种情况。$P(M=3) = 5/9$。
    

#### 第 12 题：$L = \min(X, Y)$ 分布律

- $L=1$：包含第一行和第一列，共 **5** 种情况。$P(L=1) = 5/9$。
    
- $L=2$：有 (2,2), (2,3), (3,2) 这 **3** 种情况。$P(L=2) = 3/9 = 1/3$。
    
- $L=3$：只有 (3,3) 这 **1** 种情况。$P(L=3) = 1/9$。

## 13. 二项分布之差 $Z = X - Y$

**知识点**：$B(2, 0.5)$ 的分布律为：$P(0)=0.25, P(1)=0.5, P(2)=0.25$。

### 详细步骤：

$Z$ 的可能取值为 $\{2-0, 1-0, \dots\}$ 即 $\{-2, -1, 0, 1, 2\}$。由于 $X$ 和 $Y$ 同分布，结果会对称。

- **$P(Z=-2)$**：只能是 $X=0, Y=2$。概率为 $0.25 \times 0.25 = \mathbf{0.0625}$ (即 $1/16$)。
    
- **$P(Z=-1)$**：可能是 $(0,1)$ 或 $(1,2)$。
    
    - $P(0,1) = 0.25 \times 0.5 = 0.125$
        
    - $P(1,2) = 0.5 \times 0.25 = 0.125$
        
    - 总概率 = $\mathbf{0.25}$ (即 $4/16$)。
        
- **$P(Z=0)$**：可能是 $(0,0), (1,1), (2,2)$。
    
    - $P(0,0) = 0.25^2 = 0.0625$
        
    - $P(1,1) = 0.5^2 = 0.25$
        
    - $P(2,2) = 0.25^2 = 0.0625$
        
    - 总概率 = $\mathbf{0.375}$ (即 $6/16$)。
        
- **对称性**：$P(Z=1) = P(Z=-1) = 0.25$；$P(Z=2) = P(Z=-2) = 0.0625$。

## 14. 几何分布最大值的证明（重点）

**证明思路**：利用分布函数 $F_M(k) = P(M \le k)$。

### 详细证明步骤：

1. 写出单个几何分布的分布函数：
    
    几何分布 $P(X=i) = p(1-p)^{i-1}, i=1, 2, \dots$。
    
    其分布函数 $F_X(k) = P(X \le k) = \sum_{i=1}^k p(1-p)^{i-1}$。
    
    这是一个等比数列求和，结果为：
    
    $$F_X(k) = 1 - (1-p)^k$$
    
    (物理意义：$X \le k$ 的反面是前 $k$ 次全部失败，概率为 $(1-p)^k$)
    
2. 利用最大值逻辑：
    
    $$F_M(k) = P(\max(X, Y) \le k) = P(X \le k \text{ 且 } Y \le k)$$
    
3. 利用独立性拆分：
    
    $$F_M(k) = P(X \le k) \cdot P(Y \le k)$$
    
4. 代入代数式：
    
    由于 $X, Y$ 同分布：
    
    $$F_M(k) = [F_X(k)]^2 = [1 - (1-p)^k]^2$$
    
    证毕。

### 💡 总结与启发

- **离散列表法**：当变量取值很少时（如 $3 \times 3$），画图或列表是最直观、最不容易出错的方法。
    
- **分布函数法**：处理极值（Max/Min）问题时，**永远优先考虑分布函数 $F(k)$**，而不是直接去求分布律 $P(Z=k)$。得到 $F(k)$ 后，如果需要分布律，用 $F(k) - F(k-1)$ 即可。
