这道题考查的是如何通过**概率密度函数 $f(x)$** 积分得到**分布函数 $F(x)$**。

### 1. 核心公式

根据分布函数的定义，对于连续型随机变量：

$$F(x) = P(X \leq x) = \int_{-\infty}^{x} f(t) dt$$

由于 $f(x)$ 是分段函数，我们需要分区间进行讨论计算。

---

### 2. 解题步骤

#### **第一步：当 $x \leq 0$ 时**

在这个区间内，密度函数 $f(t) = 0$。

$$F(x) = \int_{-\infty}^{x} 0 dt = 0$$

#### **第二步：当 $x > 0$ 时**

此时积分需要分为两部分：从 $-\infty$ 到 $0$（函数值为 0），以及从 $0$ 到 $x$（函数值为 $e^{-t}$）。

$$F(x) = \int_{-\infty}^{0} 0 dt + \int_{0}^{x} e^{-t} dt$$

$$F(x) = 0 + \left[ -e^{-t} \right]_{0}^{x}$$

代入上下限计算：

$$F(x) = (-e^{-x}) - (-e^{0})$$

由于 $e^{0} = 1$，所以：

$$F(x) = -e^{-x} + 1 = 1 - e^{-x}$$

---

### 3. 最终答案

将两部分合并，得到 $X$ 的分布函数 $F(x)$：

$$F(x) = \begin{cases} 0, & x \leq 0 \\ 1 - e^{-x}, & x > 0 \end{cases}$$

---

### 💡 知识点拨

- **分布函数的性质检查**：
    
    - 当 $x \to \infty$ 时，$F(x) = 1 - e^{-\infty} = 1 - 0 = 1$。
        
    - 当 $x = 0$ 时，$F(0) = 1 - e^0 = 0$，与左侧区间连续。
        
    - 这符合分布函数的基本性质（单调不减、有界、连续）。
        
- **指数分布**：这道题中的 $X$ 服从参数为 $\lambda=1$ 的**指数分布**。指数分布在可靠性工程和排队论中非常常用。
    

通过分布函数，你可以轻松计算任何区间的概率，例如 $P\{X > 1\} = 1 - F(1) = e^{-1}$。
