这组题目考查的是**离散型二维随机变量函数的分布律**。解决这类问题的通用步骤是：

1. **找出 $(X, Y)$ 所有可能的取值组合**及其对应的概率。
    
2. **计算**每种组合下函数 $Z$ 的值。
    
3. **合并**相同 $Z$ 值的概率。
    

从表中我们可以列出 $(X, Y)$ 的 6 种组合及其概率 $P(X, Y)$：

- $(-1, 0): 0.1$
    
- $(0, 0): 0.2$
    
- $(1, 0): 0.1$
    
- $(-1, 1): 0.3$
    
- $(0, 1): 0.1$
    
- $(1, 1): 0.2$
    

---

### 1. 求 $Z = X + Y$ 的分布律

计算各点的 $Z$ 值：

- $(-1, 0) \to Z = -1$
    
- $(0, 0) \to Z = 0$
    
- $(-1, 1) \to Z = 0$
    
- $(1, 0) \to Z = 1$
    
- $(0, 1) \to Z = 1$
    
- $(1, 1) \to Z = 2$
    

合并概率：

- $P(Z=-1) = 0.1$
    
- $P(Z=0) = 0.2 + 0.3 = 0.5$
    
- $P(Z=1) = 0.1 + 0.1 = 0.2$
    
- $P(Z=2) = 0.2$
    

分布律表：

| $Z$ | -1 | 0 | 1 | 2 |

| :--- | :--- | :--- | :--- | :--- |

| $P$ | 0.1 | 0.5 | 0.2 | 0.2 |

---

### 2. 求 $Z = XY$ 的分布律

计算各点的 $Z$ 值：

- $(-1, 0), (0, 0), (1, 0), (0, 1) \to Z = 0$
    
- $(-1, 1) \to Z = -1$
    
- $(1, 1) \to Z = 1$
    

合并概率：

- $P(Z=-1) = 0.3$
    
- $P(Z=0) = 0.1 + 0.2 + 0.1 + 0.1 = 0.5$
    
- $P(Z=1) = 0.2$
    

分布律表：

| $Z$ | -1 | 0 | 1 |

| :--- | :--- | :--- | :--- |

| $P$ | 0.3 | 0.5 | 0.2 |

---

### 3. 求 $Z = \max(X, Y)$ 的分布律

计算各点的 $Z$ 值：

- $(-1, 0) \to \max(-1, 0) = 0$
    
- $(0, 0) \to \max(0, 0) = 0$
    
- $(1, 0) \to \max(1, 0) = 1$
    
- $(-1, 1) \to \max(-1, 1) = 1$
    
- $(0, 1) \to \max(0, 1) = 1$
    
- $(1, 1) \to \max(1, 1) = 1$
    

合并概率：

- $P(Z=0) = 0.1 + 0.2 = 0.3$
    
- $P(Z=1) = 0.1 + 0.3 + 0.1 + 0.2 = 0.7$
    

分布律表：

| $Z$ | 0 | 1 |

| :--- | :--- | :--- |

| $P$ | 0.3 | 0.7 |

---

### 4. 求 $Z = |X - Y|$ 的分布律

计算各点的 $Z$ 值：

- $(0, 0) \to |0-0| = 0$
    
- $(1, 1) \to |1-1| = 0$
    
- $(-1, 0) \to |-1-0| = 1$
    
- $(1, 0) \to |1-0| = 1$
    
- $(0, 1) \to |0-1| = 1$
    
- $(-1, 1) \to |-1-1| = 2$
    

合并概率：

- $P(Z=0) = 0.2 + 0.2 = 0.4$
    
- $P(Z=1) = 0.1 + 0.1 + 0.1 = 0.3$
    
- $P(Z=2) = 0.3$
    

分布律表：

| $Z$ | 0 | 1 | 2 |

| :--- | :--- | :--- | :--- |

| $P$ | 0.4 | 0.3 | 0.3 |

---

### 5. 求 $Z = X^2 + Y$ 的分布律

计算各点的 $Z$ 值：

- $(0, 0) \to 0^2 + 0 = 0$
    
- $(-1, 0) \to (-1)^2 + 0 = 1$
    
- $(1, 0) \to 1^2 + 0 = 1$
    
- $(0, 1) \to 0^2 + 1 = 1$
    
- $(-1, 1) \to (-1)^2 + 1 = 2$
    
- $(1, 1) \to 1^2 + 1 = 2$
    

合并概率：

- $P(Z=0) = 0.2$
    
- $P(Z=1) = 0.1 + 0.1 + 0.1 = 0.3$
    
- $P(Z=2) = 0.3 + 0.2 = 0.5$
    

分布律表：

| $Z$ | 0 | 1 | 2 |

| :--- | :--- | :--- | :--- |

| $P$ | 0.2 | 0.3 | 0.5 |
