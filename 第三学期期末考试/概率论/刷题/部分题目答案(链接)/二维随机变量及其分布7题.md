这道题考查的是**二维均匀分布的几何概率计算**。对于均匀分布，概率的大小直接等价于**目标区域的面积与总区域面积的比值**。

以下是详细的解题步骤：

### 1. 确定总区域 $D$ 及其面积

题目给定区域 $D$ 为：$0 < x < 1, 0 < y < 1$。

这是一个在 $xy$ 平面上的单位正方形。

- **总面积 $S(D)$** = $1 \times 1 = 1$。
    
- **联合密度函数**：因为是均匀分布，$f(x, y) = \frac{1}{S(D)} = 1$（在 $D$ 内）。
    

---

### 2. 确定目标事件区域 $G$

我们需要计算 $P\{X+Y \leq 1\}$，即随机点 $(X, Y)$ 落在直线 $x + y = 1$ 下方区域的概率。

在单位正方形 $D$ 中，满足 $x + y \leq 1$（即 $y \leq 1 - x$）的区域是一个**直角三角形**：

- **顶点**：$(0, 0), (1, 0), (0, 1)$。
    
- **底边长** = $1$。
    
- **高** = $1$。
    

---

### 3. 计算概率

根据几何概率的定义：

$$P\{X+Y \leq 1\} = \frac{\text{目标区域 } G \text{ 的面积}}{\text{总区域 } D \text{ 的面积}}$$

1. 目标面积 $S(G)$：
    
    $$S(G) = \frac{1}{2} \times \text{底} \times \text{高} = \frac{1}{2} \times 1 \times 1 = \frac{1}{2}$$
    
2. 计算概率：
    
    $$P\{X+Y \leq 1\} = \frac{1/2}{1} = \frac{1}{2}$$
    

---

### 4. 最终答案

$$P\{X+Y \leq 1\} = 0.5$$

---

### 💡 知识拓展：如果改用积分法？

虽然几何法最快，但你也可以用二重积分验证：

$$P = \iint_{x+y \leq 1} 1 \, dxdy = \int_{0}^{1} \left( \int_{0}^{1-x} 1 \, dy \right) dx = \int_{0}^{1} (1-x) dx = \left[ x - \frac{1}{2}x^2 \right]_0^1 = \frac{1}{2}$$

### 考试避坑小贴士

- **看清范围**：如果区域 $D$ 变成了 $0 < x < 2, 0 < y < 2$，那么总面积就是 4，分母会变。
    
- **看清符号**：如果是 $P\{X+Y > 1\}$，那就是求正方形中剩余的那个三角形（右上角部分），概率同样是 $1 - 0.5 = 0.5$。
