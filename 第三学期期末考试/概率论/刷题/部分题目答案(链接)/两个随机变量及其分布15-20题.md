### 15. 条件下的函数分布

> **题目：** 设 $(X, Y)$ 分布律同前表，求在 $Y=1$ 条件下，$Z=X^2$ 的分布律。

- **核心知识点：** 条件分布的“局部缩放” + 函数映射。
    
- 第一步：求条件分布 $P(X|Y=1)$
    
    查阅之前（第1-5题）的表格，当 $Y=1$ 时，$X$ 的行概率之和为 $0.3 + 0.1 + 0.2 = 0.6$。
    
    - $P(X=-1 | Y=1) = 0.3 / 0.6 = 0.5$
        
    - $P(X=0 | Y=1) = 0.1 / 0.6 = 1/6$
        
    - $P(X=1 | Y=1) = 0.2 / 0.6 = 1/3$
        
- **第二步：映射到 $Z=X^2$**
    
    - 当 $X=-1$ 时，$Z=1$
        
    - 当 $X=0$ 时，$Z=0$
        
    - 当 $X=1$ 时，$Z=1$
        
- **第三步：合并概率**
    
    - $P(Z=0) = 1/6$
        
    - $P(Z=1) = 0.5 + 1/3 = 5/6$
        
- **结论：** $Z$ 的取值为 $\{0, 1\}$，对应概率为 $\{1/6, 5/6\}$。

### 16. 二项分布的函数映射

> **题目：** $X \sim B(3, 0.5)$，求 $Y = X^2 - 1$ 的分布律。

- **核心知识点：** 二项分布公式 $P(X=k) = \binom{n}{k} p^k (1-p)^{n-k}$。
    
- **第一步：算 $X$ 的概率** ($n=3, p=0.5$)
    
    - $P(X=0) = 1/8$
        
    - $P(X=1) = 3/8$
        
    - $P(X=2) = 3/8$
        
    - $P(X=3) = 1/8$
        
- **第二步：算 $Y$ 的取值**
    
    - $X=0 \to Y=-1$；$X=1 \to Y=0$；$X=2 \to Y=3$；$X=3 \to Y=8$
        
- 结论： $Y$ 的分布律为：
    
    | $Y$ | -1 | 0 | 3 | 8 |
    
    | :--- | :--- | :--- | :--- | :--- |
    
    | $P$ | 1/8 | 3/8 | 3/8 | 1/8 |

### 17. 指示函数（判别分布）

> **题目：** $X, Y$ 独立且取值 $\{0, 1\}$。$P(X=1)=p, P(Y=1)=q$。定义 $Z = \mathbb{I}_{\{X=Y\}}$（相等取1，否则取0），求 $Z$ 的分布律。

- **核心知识点：** 指示函数的定义。$Z$ 只有两种可能：$1$（代表相等）或 $0$（代表不等）。
    
- 计算 $P(Z=1)$（即 $X=Y$ 的概率）：
    
    由于独立，相等只有两种情况：都是0或都是1。
    
    - $P(X=0, Y=0) = (1-p)(1-q)$
        
    - $P(X=1, Y=1) = pq$
        
    - $P(Z=1) = (1-p)(1-q) + pq = 1 - p - q + 2pq$
        
- **计算 $P(Z=0)$：** $1 - P(Z=1) = p + q - 2pq$。

### 18. 泊松-二项混合分布（高频考点）

> **题目：** $X \sim P(\lambda)$，已知 $X=n$ 时，$Y \sim B(n, p)$。求 $Y$ 的边缘分布律。

- **核心知识点：** 全概率公式的离散应用。$P(Y=k) = \sum P(Y=k|X=n)P(X=n)$。
    
- **推导简述：**
    
    1. 代入公式：$P(Y=k) = \sum_{n=k}^{\infty} \left[ \binom{n}{k} p^k (1-p)^{n-k} \right] \cdot \left[ \frac{e^{-\lambda} \lambda^n}{n!} \right]$
        
    2. 利用组合数展开和指数级数 $\sum \frac{x^m}{m!} = e^x$ 进行化简。
        
- **结论：** 这是一个非常著名的结论：**$Y \sim P(\lambda p)$**。
    
    > **直观理解：** 如果进店人数服从泊松分布，每个人买东西的概率是 $p$，那么买东西的总人数依然服从泊松分布，均值变为原来的 $p$ 倍。

### 19. 离散均匀分布的相遇问题

> **题目：** $X, Y$ 独立服从 $\{1, 2, \dots, N\}$ 上的均匀分布，求 $P(X=Y)$。

- **核心知识点：** 独立变量的联合概率 $P(X=i, Y=j) = P(X=i)P(Y=j)$。
    
- **计算：**
    
    - 任意一个点 $(i, i)$ 的概率都是 $\frac{1}{N} \times \frac{1}{N} = \frac{1}{N^2}$。
        
    - 共有 $N$ 个这样的点（即对角线上的点）。
        
- **结论：** $P(X=Y) = N \times \frac{1}{N^2} = \frac{1}{N}$。

### 20. 泊松分布的条件分布（重点证明题）

> **题目：** $X, Y$ 独立服从参数为 $\lambda$ 的泊松分布，求 $P(X=k | X+Y=n)$。

- **核心知识点：** 条件概率公式 + 第一张图片中的“泊松分布可加性”。
    
- **推导过程：**
    
    1. 公式：$P(X=k | X+Y=n) = \frac{P(X=k, Y=n-k)}{P(X+Y=n)}$
        
    2. 分子：由于独立，等于 $P(X=k) \cdot P(Y=n-k)$
        
    3. 分母：根据第一张图的结论，$X+Y \sim P(2\lambda)$
        
    4. 代入泊松公式并约分：
        
        $$\frac{\frac{e^{-\lambda}\lambda^k}{k!} \cdot \frac{e^{-\lambda}\lambda^{n-k}}{(n-k)!}}{\frac{e^{-2\lambda}(2\lambda)^n}{n!}} = \frac{n!}{k!(n-k)!} \cdot \frac{\lambda^n}{2^n \lambda^n} = \binom{n}{k} \left(\frac{1}{2}\right)^n$$
        
- **结论：** 该分布是 **二项分布 $B(n, 0.5)$**。
    
    > **物理意义：** 已知两台机器总共出了 $n$ 个故障，且两台机器故障率相同，那么其中一台机器贡献 $k$ 个故障的概率服从二项分布。
