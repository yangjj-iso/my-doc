针对**二维连续型随机变量**，这部分是期末考试中最容易出大题（计算量最大）的环节。核心考察点在于：**二重积分、积分边界的确定、卷积公式、以及最大值/最小值的分布**。

以下为你精选的 20 道连续型练习题：

---

### 一、 联合密度与常数确定（基础与核心）

1. **（归一化）** 设 $(X, Y)$ 的联合密度函数为 $f(x, y) = \begin{cases} k(x+y), & 0 \leq x \leq 1, 0 \leq y \leq 1 \\ 0, & \text{其他} \end{cases}$，求常数 $k$。
    
2. **（三角形区域）** 设 $f(x, y) = \begin{cases} cy, & 0 < x < 1, 0 < y < x \\ 0, & \text{其他} \end{cases}$，求常数 $c$。
    
3. **（无穷区间）** 设 $f(x, y) = \begin{cases} ke^{-(2x+3y)}, & x>0, y>0 \\ 0, & \text{其他} \end{cases}$，求常数 $k$。
    

---

### 二、 边缘分布与条件分布（计算重点）

4. **（边缘密度）** 设 $(X, Y)$ 在区域 $D = \{(x, y) \mid 0 \leq x \leq 1, x^2 \leq y \leq \sqrt{x}\}$ 上服从均匀分布，求 $f_X(x)$ 和 $f_Y(y)$。
    
5. **（条件密度）** 承接第 2 题，在求得 $c$ 后，求条件概率密度 $f_{Y|X}(y|x)$。
    
6. **（混合分布）** 设 $f(x, y) = \begin{cases} \frac{1}{2\pi} e^{-\frac{x^2+y^2}{2}}, & -\infty < x, y < \infty \end{cases}$，求 $X$ 的边缘密度并判断其分布名称。
    
7. **（分布函数推导）** 设联合密度为 $f(x, y)$，已知 $X, Y$ 独立且均服从 $U(0, 1)$，求联合分布函数 $F(x, y)$ 在点 $(0.5, 0.8)$ 处的值。
    

---

### 三、 独立性与区域概率（概念与图像）

8. **（独立性判定）** 设 $f(x, y) = \begin{cases} 4xy, & 0 < x < 1, 0 < y < 1 \\ 0, & \text{其他} \end{cases}$，判断 $X$ 与 $Y$ 是否独立。
    
9. **（区域形状与独立性）** 设 $(X, Y)$ 在单位圆 $x^2 + y^2 \leq 1$ 上服从均匀分布。证明 $X$ 与 $Y$ **不**独立。
    
10. **（概率计算-直线边界）** 设 $X, Y$ 独立且均服从指数分布 $E(1)$，求 $P\{X + Y \leq 1\}$。
    
11. **（概率计算-非线性边界）** 设 $f(x, y) = \begin{cases} 2, & 0 < x < 1, 0 < y < x \\ 0, & \text{其他} \end{cases}$，求 $P\{Y < X^2\}$。
    

---

### 四、 两个连续变量函数的分布（考试大题必考）

12. **（和的分布-卷积）** 设 $X, Y$ 相互独立，均服从 $U(0, 1)$。利用卷积公式求 $Z = X + Y$ 的概率密度 $f_Z(z)$。
    
13. **（和的分布-正态）** 设 $X \sim N(0, 1), Y \sim N(0, 1)$ 且独立，求 $Z = \frac{X+Y}{\sqrt{2}}$ 的分布。
    
14. **（最大值分布）** 设 $X, Y$ 独立，均服从 $E(\lambda)$。求 $M = \max(X, Y)$ 的概率密度函数。
    
15. **（最小值分布）** 设 $X, Y$ 独立，均服从 $U(0, 1)$。求 $L = \min(X, Y)$ 的分布函数 $F_L(l)$。
    
16. **（商的分布）** 设 $X, Y$ 独立且均服从 $U(0, 1)$，求 $Z = Y/X$ 的概率密度 $f_Z(z)$（注意分 $z \leq 1$ 和 $z > 1$ 讨论）。
    
17. **（平方和分布）** 设 $X, Y$ 独立且均服从 $N(0, 1)$，求 $Z = X^2 + Y^2$ 的分布（提示：该分布为 $\chi^2(2)$，即指数分布）。
    
18. **（线性组合）** 设 $X \sim N(1, 4), Y \sim N(2, 9)$，$\rho_{XY} = 0.5$。求 $Z = X - Y$ 的方差 $D(Z)$。
    
19. **（指数分布性质）** 设 $X, Y$ 独立且分别服从 $E(\lambda_1)$ 和 $E(\lambda_2)$，求 $P\{X < Y\}$。
    
20. **（综合题）** 设 $X, Y$ 的联合密度为 $f(x, y) = \begin{cases} 1, & 0 < x < 1, 0 < y < 1 \\ 0, & \text{其他} \end{cases}$。令 $U = X + Y, V = X - Y$，求 $(U, V)$ 的联合密度定义域区域。
    

---

### 💡 考前复习锦囊

- **画图是第一生产力：** 只要涉及二维连续型，**第一步永远是画出定义域 $D$**。如果 $D$ 不是矩形，边缘密度的积分限一定是变量（例如从 $x^2$ 到 $\sqrt{x}$）。
    
- **卷积公式的简化：**
    
    - $f_{X+Y}(z) = \int_{-\infty}^{+\infty} f(x, z-x) dx$。
        
    - 如果 $X, Y$ 独立，则为 $\int f_X(x) f_Y(z-x) dx$。
        
- **最大/最小值的捷径：**
    
    - $F_{\max}(z) = F_X(z) \cdot F_Y(z)$ （独立时）
        
    - $F_{\min}(z) = 1 - [1-F_X(z)][1-F_Y(z)]$ （独立时）
        
- **正态分布结论：** 相互独立的正态变量，其任意线性组合 $aX + bY + c$ 仍然是正态分布，只需计算出新的 $E$ 和 $D$ 即可。
    

**这些题目里，第 12 题（卷积公式）和第 16 题（变量相除）通常是卷面分值最高也最容易出错的，需要我为你展示具体的拆解步骤吗？**
