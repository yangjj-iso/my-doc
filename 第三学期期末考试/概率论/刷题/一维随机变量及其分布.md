### 一、 概念与离散型随机变量

1. **（性质）** 设随机变量 $X$ 的分布律为 $P\{X=k\} = a \lambda^k$ ($k=1, 2, ...$)，则常数 $a$ 应满足什么条件？[[一维随机变量及其分布1题]]
    
2. **（分布函数）** 设 $F(x)$ 是随机变量 $X$ 的分布函数，下列结论中正确的是： D[[一维随机变量及其分布2题]]
    
    - A. $F(x)$ 必是连续函数
        
    - B. $P\{X=x\} = F(x) - F(x-0)$
        
    - C. $F(x)$ 必是严格单调增加的
        
    - D. $F(x) \geq 0$ 且 $\lim_{x \to \infty} F(x) = 1$
        
3. **（二项分布）** 某射手每次射击中靶的概率为 0.8，现独立射击 5 次，求至少中靶 2 次的概率。
    $$P(X \ge 2) = 1 - 0.00672 = 0.99328$$
4. **（泊松分布）** 设 $X \sim P(\lambda)$，且 $P\{X=1\} = P\{X=2\}$，求 $P\{X=4\}$。 OK[[一维随机变量及其分布4题]]
    
5. **（离散型性质）** 设随机变量 $X$ 只取 $0, 1, 2$ 三个值，其概率依次为 $\frac{1}{2}, \frac{1}{3}, \frac{1}{6}$，求 $X$ 的分布函数 $F(x)$。
    
6. **（古典概型+离散）** 袋中有 3 个白球 2 个黑球，从中无放回地任取 2 个，记 $X$ 为取出的白球数，求 $X$ 的分布律。

### 二、 连续型随机变量与密度函数

7. **（性质）** 已知随机变量 $X$ 的概率密度为 $f(x) = \begin{cases} cx^2, & 0 < x < 3 \\ 0, & \text{其他} \end{cases}$，求常数 $c$。[[一维随机变量及其分布7题]]
    
8. **（概率计算）** 承接上题，求 $P\{1 < X < 2\}$。
    $P\{1 < X < 2\} = \frac{7}{27} \approx 0.259$。
9. **（分布函数推导）** 设 $X$ 的密度函数为 $f(x) = \begin{cases} e^{-x}, & x > 0 \\ 0, & x \leq 0 \end{cases}$，求其分布函数 $F(x)$。[[一维随机变量及其分布9题]]
    
10. **（均匀分布）** 设 $X \sim U(2, 5)$，现对 $X$ 进行独立观测 3 次，求至少有 2 次观测值大于 3 的概率。
    
11. **（指数分布）** 设某元件的使用寿命 $X$（单位：小时）服从参数为 $\lambda=1/1000$ 的指数分布，求该元件能使用超过 2000 小时的概率。
    
12. **（无记忆性）** 证明：若 $X$ 服从指数分布，则对于任意 $s, t > 0$，有 $P\{X > s+t \mid X > s\} = P\{X > t\}$。[[一维随机变量及其分布12题]]
    
13. **（密度函数识别）** 设 $F(x) = \begin{cases} 0, & x < 0 \\ x/2, & 0 \leq x < 1 \\ 1, & x \geq 1 \end{cases}$，问 $X$ 是否为连续型随机变量？如果是，求其密度函数。
    不是,主要是判断区间内函数是否连续,不连续就不是连续型随机变量

### 三、 正态分布（考试重点）

14. **（标准化）** 设 $X \sim N(3, 2^2)$，求 $P\{2 < X < 5\}$。（结果用标准正态分布函数 $\Phi$ 表示）
    **$\Phi(1) - \Phi(-0.5)$** 或 **$\Phi(1) + \Phi(0.5) - 1$**。
15. **（对称性）** 设 $X \sim N(\mu, \sigma^2)$，若 $P\{X < 2\} = P\{X > 4\}$，求参数 $\mu$。
    $\mu = 3$
16. **（标准正态性质）** 设 $Z \sim N(0, 1)$，已知 $\Phi(1.96) = 0.975$，求 $P\{|Z| > 1.96\}$。
    $P\{|Z| > 1.96\} = 0.05$
17. **（分位数）** 设 $X \sim N(10, 4)$，若 $P\{X > a\} = 0.05$，如何利用标准正态分布表求 $a$？
    $$a = 13.29$$

### 四、 随机变量函数的分布

18. **（线性变换）** 设 $X \sim N(\mu, \sigma^2)$，证明 $Y = aX + b$ ($a \neq 0$) 也服从正态分布，并写出其均值和方差。
    
19. **（离散型函数）** 设 $X$ 的分布律为 $P\{X=-1\} = 0.2, P\{X=0\} = 0.3, P\{X=1\} = 0.5$，求 $Y = X^2$ 的分布律。
    
20. **（连续型函数/单调）** 设 $X$ 在 $(0, 1)$ 上服从均匀分布，求 $Y = -2 \ln X$ 的概率密度函数。

### 💡 复习建议与部分简答

1. **归一化性质：** 无论离散（$\sum p_i = 1$）还是连续（$\int f(x)dx = 1$），这是求常数 $c$ 或 $k$ 的必考点。
    
2. **分布函数 $F(x)$：** 它是**累积**概率 $P\{X \leq x\}$。求分段函数的 $F(x)$ 时，一定要注意分段讨论，且 $F(x)$ 在全线右连续。
    
3. **正态分布查表：** 记住公式 $P\{X \leq x\} = \Phi(\frac{x-\mu}{\sigma})$。
    
4. **第 20 题参考答案：** $Y$ 服从参数为 $1/2$ 的指数分布。这是生成随机数常用的“逆变换法”原理。
