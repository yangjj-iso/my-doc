标题提到的“2种题型”**，在期末考试中通常指的是：

1. **单正态总体的检验**（只看一个班的成绩是否达标）。
    
2. 双正态总体的检验（比较两个班的成绩是否有差异）。
    
    (当然，最核心的还是围绕 均值 $\mu$ 和 方差 $\sigma^2$ 的检验)

### 一、 解题核心：五步通关法 (必背模版)

假设检验的逻辑就像法院判案：先假设嫌疑人无罪 ($H_0$)，除非有确凿证据 ($H_1$) 证明他有罪。

**标准解题步骤：**

1. **提出假设：**
    
    - **原假设 $H_0$：** 总是包含“等于号” ($=, \ge, \le$)。通常是我们想要拒绝或维持现状的假设。
        
    - **备择假设 $H_1$：** 包含“不等号” ($\neq, >, <$)。这是我们想证明的结论。
        
2. **选统计量：**
    
    - 和区间估计一模一样！($\sigma$ 已知用 $Z$，$\sigma$ 未知用 $t$，方差用 $\chi^2$)。
        
3. **定拒绝域：**
    
    - 根据 $\alpha$ (显著性水平) 和 $H_1$ 的符号方向，查表画出“红线”。
        
4. **计算数值：**
    
    - 把样本数据代入公式算出统计量的值。
        
5. **下结论：**
    
    - 数值落入拒绝域 $\rightarrow$ 拒绝 $H_0$，接受 $H_1$。
        
    - 数值没落入拒绝域 $\rightarrow$ 接受 $H_0$。

### 二、 核心考点：如何选公式？(跟上一章完全通用)

主要考 **均值 $\mu$ 的检验** (Z检验与t检验)。

|**情况**|**检验统计量公式**|**H1​ 的符号**|**拒绝域 (落在这里就拒绝)**|
|---|---|---|---|
|**$\sigma$ 已知**|$Z = \frac{\bar{X} - \mu_0}{\sigma / \sqrt{n}}$|$\neq$ (双侧)|$\\|Z\\| \ge z_{\alpha/$|
|(用 Z)||$>$ (单侧)|$Z \ge z_{\alpha}$|
|||$<$ (单侧)|$Z \le -z_{\alpha}$|
|**$\sigma$ 未知**|$T = \frac{\bar{X} - \mu_0}{S / \sqrt{n}}$|$\neq$ (双侧)|$\\|T\\| \ge t_{\alpha/2}(n-$|

_注：$\sigma$ 未知时，记得算样本标准差 $S$。_

### 三、 经典例题与实战技巧

#### 例题 1：均值检验 (双侧，$\sigma$ 未知)

题目：

某工厂生产的零件规定平均长度为 10cm。现从一批产品中随机抽取 $n=16$ 个，测得平均长度 $\bar{x} = 9.92$cm，样本标准差 $s = 0.2$cm。在 $\alpha=0.05$ 水平下，检验这批零件是否合格？(认为长度服从正态分布，且 $t_{0.025}(15) = 2.131$)

**解题演示 (五步法)：**

1. 提出假设：
    
    题目问“是否合格”（即是否等于10），所以是双侧检验。
    
    $H_0: \mu = 10$ (合格)
    
    $H_1: \mu \neq 10$ (不合格)
    
2. 选统计量：
    
    题目没给总体标准差 $\sigma$，只给了样本标准差 $s$，所以用 t检验。
    
    统计量 $T = \frac{\bar{X} - \mu_0}{S / \sqrt{n}}$
    
3. 定拒绝域：
    
    双侧检验，$\alpha = 0.05$，查表 $t_{\alpha/2}(n-1) = t_{0.025}(15) = 2.131$。
    
    拒绝域为：$|T| \ge 2.131$。
    
4. 计算：
    
    $$T = \frac{9.92 - 10}{0.2 / \sqrt{16}} = \frac{-0.08}{0.05} = -1.6$$
    
5. 结论：
    
    因为 $|T| = 1.6 < 2.131$，统计量没有落入拒绝域。
    
    所以接受 $H_0$，认为这批零件是合格的。

#### 技巧点拨：如何判断“单侧”还是“双侧”？

这是考试最大的坑，$H_1$ 设反了，全盘皆输。

- **双侧检验 ($\neq$)：** 题目出现 **“是否有显著差异”、“是否改变”、“是否准确”**。
    
    - _例：检验机器是否正常运行？（太大太小都不行）_
        
- **单侧检验 ($> 或 <$)：** 题目出现 **“是否提高”、“是否降低”、“优于”、“劣于”**。
    
    - _例：采用了新工艺，强度是否**提高**了？ ($H_1: \mu > \mu_0$)_
        
    - _例：废品率是否**降低**了？ ($H_1: \mu < \mu_0$)_
        

**⚡️ 避坑口诀：**

> “小概率事件”放 $H_1$。
> 
> 我们通常想证明新工艺有效、新药有效，所以把“有效”（有变化）放在 $H_1$。$H_0$ 永远放那个包含“等号”的（比如 $\mu \le \mu_0$ 或 $\mu = \mu_0$）。

### 四、 总结与考前建议

1. **参数估计 vs 假设检验：** 它们是一枚硬币的两面。置信区间 $[\bar{X} - \Delta, \bar{X} + \Delta]$ 其实就是接受域。如果 $\mu_0$ 在置信区间内，就接受 $H_0$；如果在外面，就拒绝。
    
2. **必考题型预测：** 期末考试 90% 会考一道 **“$\sigma$ 未知，单正态总体均值检验”** 的大题（就像上面的例题）。它考察了你算 $S$ 的能力、查 $t$ 分布表的能力、以及写出五步法的逻辑。
