这一章在期末考试中通常是一道**分值巨大的计算大题**（10-15分），而且步骤非常死板，属于**“只要背了流程就一定能拿满分”**的类型。

标题中的“3种题型”**，对应的就是参数估计的三大金刚：

1. **矩估计 (Method of Moments)** —— 最简单，解方程。
    
2. **最大似然估计 (MLE)** —— 最重要，求导数。
    
3. **区间估计 (Interval Estimation)** —— 最死板，查公式。
    

下面我为你拆解这三种题型的**标准解题模版**和**秒杀技巧**。

---

### 题型一：矩估计 (点估计之“简单粗暴法”)

核心思想：

样本的平均值 $\bar{X}$ 应该等于总体的期望 $E(X)$。

**解题三步走（万能模版）：**

1. **算期望：** 根据题目给的密度函数，算出总体期望 $E(X)$（结果通常包含未知参数 $\theta$）。
    
2. **列方程：** 令 $E(X) = \bar{X}$。
    
3. **解参数：** 把 $\theta$ 反解出来，加上帽子 $\hat{\theta}$，结束。
    

⚡️ 经典例题：

设总体 $X$ 服从 $[0, \theta]$ 上的均匀分布，求 $\theta$ 的矩估计量。

> **解：**
> 
> 1. 总体期望 $E(X) = \frac{0 + \theta}{2} = \frac{\theta}{2}$。
>     
> 2. 令 $E(X) = \bar{X}$，即 $\frac{\theta}{2} = \bar{X}$。
>     
> 3. 解得 $\hat{\theta} = 2\bar{X}$。
>     
>     (完。是不是简单得离谱？)
>     

**⚠️ 进阶坑点：** 如果有两个参数（比如 $\mu, \sigma^2$），你需要列两个方程：

1. $E(X) = \bar{X}$
    
2. $E(X^2) = \frac{1}{n}\sum X_i^2$ （或者直接用 $D(X) = \sigma^2$ 对应二阶中心矩 $S^2$）

### 题型二：最大似然估计 (MLE) —— 必考！必考！

这是期末考试**出镜率最高**的题型。

**解题四步走（标准套路）：**

1. 写似然函数 $L(\theta)$：
    
    把 $n$ 个密度函数乘起来：$L(\theta) = \prod_{i=1}^n f(x_i; \theta)$。
    
2. 取对数 $\ln L$：
    
    把乘积变成求和（为了好求导）：$\ln L(\theta) = \sum_{i=1}^n \ln f(x_i; \theta)$。
    
3. 求导置零：
    
    对 $\theta$ 求导，并令导数为 0：$\frac{d \ln L}{d\theta} = 0$。
    
4. 求解：
    
    解出 $\theta$，即为 $\hat{\theta}$。
    

⚡️ 经典例题：

设总体 $X$ 服从参数为 $\lambda$ 的指数分布 $f(x) = \lambda e^{-\lambda x} (x>0)$，求 $\lambda$ 的最大似然估计。

> **解：**
> 
> 1. 构造似然函数：
>     
>     $$L(\lambda) = \prod_{i=1}^n (\lambda e^{-\lambda x_i}) = \lambda^n e^{-\lambda \sum x_i}$$
>     
> 2. 取对数：
>     
>     $$\ln L = n \ln \lambda - \lambda \sum_{i=1}^n x_i$$
>     
> 3. 求导并令为0：
>     
>     $$\frac{d \ln L}{d \lambda} = \frac{n}{\lambda} - \sum_{i=1}^n x_i = 0$$
>     
> 4. 求解：
>     
>     $$\hat{\lambda} = \frac{n}{\sum x_i} = \frac{1}{\bar{X}}$$
>     

☠️ 唯一的“陷阱” (均匀分布)：

如果题目是 $U(0, \theta)$ 这种范围里包含参数的分布，千万不能求导！

- **技巧：** $L(\theta)$ 是增函数还是减函数？
    
- **结论：** 均匀分布 $[0, \theta]$ 的最大似然估计直接写：$\hat{\theta} = \max(X_1, ..., X_n)$ （样本里的最大值）。

### 题型三：区间估计 (置信区间)

这道题不需要动脑子，需要的是**记忆力**和**视力**。你要根据题目条件，选对公式。

**主要考正态总体均值 $\mu$ 的估计，分两种情况：**

| **情况**     | **已知条件**        | **选用的统计量 (枢轴量)** | **什么时候用？**          |
| ---------- | --------------- | ---------------- | ------------------- |
| **Case 1** | **$\sigma$ 已知** | **$Z$ (标准正态)**   | 题目直接给了方差/标准差具体的数字   |
| **Case 2** | **$\sigma$ 未知** | **$t$ (t分布)**    | 题目只给了样本数据，让你自己算 $S$ |

**解题公式 (背下来)：**

1. $\sigma$ 已知（用 $Z$）：
    
    $$(\bar{X} - z_{\alpha/2} \frac{\sigma}{\sqrt{n}}, \quad \bar{X} + z_{\alpha/2} \frac{\sigma}{\sqrt{n}})$$
    
2. $\sigma$ 未知（用 $t$）： (期末最爱考这个，因为让你多算一个 $S$)
    
    $$(\bar{X} - t_{\alpha/2}(n-1) \frac{S}{\sqrt{n}}, \quad \bar{X} + t_{\alpha/2}(n-1) \frac{S}{\sqrt{n}})$$
    

**做题步骤：**

1. 算出样本均值 $\bar{X}$。
    
2. (如果 $\sigma$ 未知) 算出样本标准差 $S = \sqrt{\frac{\sum(x_i-\bar{X})^2}{n-1}}$。
    
3. 根据置信水平 $1-\alpha$ (比如95%)，去查表找到 $z_{\alpha/2}$ 或 $t_{\alpha/2}$。
    
4. 代入公式。
