这是一个在概率论与数理统计中非常经典且核心的问题。

求解两个连续型随机变量 $X$ 和 $Y$ 的函数 $Z = g(X, Y)$ 的分布（通常求概率密度函数 PDF），主要有两种通用的方法：**分布函数法（定义法）** 和 **卷积公式法**（主要针对和、差、积、商）。![[Pasted image 20251220155119.png]]

### 一、 通用方法：分布函数法 (CDF Method)

这是最基础也最通用的方法，适用于任何连续函数关系 $Z = g(X, Y)$。

**核心思路：** 先求 $Z$ 的累积分布函数 $F_Z(z)$，再对 $z$ 求导得到概率密度函数 $f_Z(z)$。

**步骤：**

1. 定义： 根据分布函数的定义写出表达式：
    
    $$F_Z(z) = P(Z \le z) = P(g(X, Y) \le z)$$
    
2. 积分： 将上述概率转化为二重积分。假设 $X, Y$ 的联合概率密度为 $f(x, y)$，积分区域 $D_z$ 由不等式 $g(x, y) \le z$ 确定：
    
    $$F_Z(z) = \iint_{D_z} f(x, y) \,dx\,dy$$
    
3. 求导： 对 $F_Z(z)$ 关于 $z$ 求导，即可得到 $Z$ 的概率密度函数：
    
    $$f_Z(z) = F'_Z(z)$$
    

> **注意：** 如果 $X$ 和 $Y$ 相互独立，则 $f(x, y) = f_X(x) \cdot f_Y(y)$。

---

### 二、 常见运算的公式 (Formula Method)

对于常见的四则运算（和、差、积、商），我们可以直接使用由分布函数法推导出来的公式。

假设 $X$ 和 $Y$ 是连续型随机变量，联合密度为 $f(x, y)$。

#### 1. 和的分布 ($Z = X + Y$) —— 卷积公式

这是最重要的公式。

- 一般情况：
    
    $$f_Z(z) = \int_{-\infty}^{\infty} f(x, z-x) \,dx$$
    
- 若 $X, Y$ 独立：
    
    $$f_Z(z) = \int_{-\infty}^{\infty} f_X(x) f_Y(z-x) \,dx$$
    
    这被称为 $f_X$ 和 $f_Y$ 的卷积 (Convolution)，记作 $f_X * f_Y$。
    

#### 2. 差的分布 ($Z = X - Y$)

- 一般情况：
    
    $$f_Z(z) = \int_{-\infty}^{\infty} f(x, x-z) \,dx$$
    
- 若 $X, Y$ 独立：
    
    $$f_Z(z) = \int_{-\infty}^{\infty} f_X(x) f_Y(x-z) \,dx$$
    

#### 3. 积的分布 ($Z = XY$)

- 若 $X, Y$ 独立：
    
    $$f_Z(z) = \int_{-\infty}^{\infty} \frac{1}{|x|} f_X(x) f_Y\left(\frac{z}{x}\right) \,dx$$
    

#### 4. 商的分布 ($Z = \frac{X}{Y}$)

- 若 $X, Y$ 独立：
    
    $$f_Z(z) = \int_{-\infty}^{\infty} |y| f_X(yz) f_Y(y) \,dy$$

### 三、 经典示例：两个均匀分布之和

为了让你更直观地理解，我们来看一个经典例子。

**题目：** 设 $X, Y$ 相互独立，且都服从 $[0, 1]$ 上的均匀分布。求 $Z = X + Y$ 的概率密度函数。

**分析：**

- $f_X(x) = 1, \quad 0 \le x \le 1$
    
- $f_Y(y) = 1, \quad 0 \le y \le 1$
    
- $Z$ 的取值范围是 $[0, 2]$。
    

利用卷积公式求解：

$$f_Z(z) = \int_{-\infty}^{\infty} f_X(x) f_Y(z-x) \,dx$$

由于 $f_X(x)$ 和 $f_Y(z-x)$ 非零的条件分别是 $0 \le x \le 1$ 和 $0 \le z-x \le 1$（即 $z-1 \le x \le z$）。

我们需要计算这两个区间的交集长度。

1. 当 $0 < z \le 1$ 时：
    
    积分区间为 $[0, z]$。
    
    $$f_Z(z) = \int_{0}^{z} 1 \cdot 1 \,dx = z$$
    
2. 当 $1 < z < 2$ 时：
    
    积分区间为 $[z-1, 1]$。
    
    $$f_Z(z) = \int_{z-1}^{1} 1 \cdot 1 \,dx = 1 - (z-1) = 2 - z$$
    
3. **其他情况：** $f_Z(z) = 0$。
    

结果： 这是一个三角形分布 (Triangular Distribution)。

$$f_Z(z) = \begin{cases} z, & 0 < z \le 1 \\ 2-z, & 1 < z < 2 \\ 0, & \text{其他} \end{cases}$$

### 四、 重要特例：正态分布的可加性

在实际应用（如工程、金融）中，最重要的结论是关于正态分布的：

如果 $X \sim N(\mu_1, \sigma_1^2)$， $Y \sim N(\mu_2, \sigma_2^2)$，且 $X, Y$ 相互独立。

那么它们的线性组合 $Z = aX + bY$ 仍然服从正态分布：

$$Z \sim N(a\mu_1 + b\mu_2, a^2\sigma_1^2 + b^2\sigma_2^2)$$

这是一个非常有用的性质，因为你不需要每次都去积分，直接套用参数即可。
