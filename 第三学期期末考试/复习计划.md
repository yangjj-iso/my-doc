### 三、 给你的特别“数据库 SQL”特训包

既然 SQL 是你的痛点，而且现在时间充裕了，建议每天拿出 **30分钟** 进行“肌肉记忆训练”。

**这是你要死磕的 SQL 3大难点（搞定这3个，期末及格没问题）：**

1. **分组陷阱：**
    
    - 题目出现“每门课”、“每个班”、“每位同学”，马上反应过来用 `GROUP BY`。
        
    - _口诀：_ `SELECT` 后面出现的列，如果没有被聚合函数（sum, avg, count）包围，就必须出现在 `GROUP BY` 后面。
        
2. **多表连接（必考）：**
    
    - 搞不懂 `INNER JOIN` / `LEFT JOIN`？期末考通常只需要用 `WHERE` 暴力连接：
        
    - `SELECT T.name FROM Teacher T, Course C WHERE T.id = C.tid AND C.name = '数据库'` （虽然老土，但能得分！）
        
3. **去重与排序：**
    
    - 问“有哪些系”，记得 `SELECT DISTINCT 系名 ...`
        
    - 问“前三名”，记得 `ORDER BY grade DESC` (部分数据库还要加 `LIMIT 3`，看你们教材教的是哪种方言，通常是 MySQL 或 SQL Server)。
