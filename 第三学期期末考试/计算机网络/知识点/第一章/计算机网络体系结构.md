## 1. 三种主要的体系结构对比

目前主流的有三种模型，考试最常考的是**五层原理体系结构**（因为它最适合教学且贴合互联网实际）。

|**层次**|**OSI 七层模型（法律标准）**|**TCP/IP 四层模型（事实标准）**|**五层原理体系结构（教学推荐）**|
|---|---|---|---|
|**第 7 层**|**应用层** (Application)|同样合并在应用层|**应用层**|
|**第 6 层**|**表示层** (Presentation)|(TCP/IP 中由应用层处理)|(合并在应用层)|
|**第 5 层**|**会话层** (Session)|(TCP/IP 中由应用层处理)|(合并在应用层)|
|**第 4 层**|**运输层** (Transport)|**运输层**|**运输层**|
|**第 3 层**|**网络层** (Network)|**网际层** (Internet)|**网络层**|
|**第 2 层**|**数据链路层** (Data Link)|同样合并在网络接口层|**数据链路层**|
|**第 1 层**|**物理层** (Physical)|**网络接口层** (Network Interface)|**物理层**|

## 2. 五层协议体系结构详解 (核心考点)

### ① 应用层 (Application Layer)

- **任务：** 通过应用进程间的交互来完成特定网络应用。
    
- **数据单位：** **报文 (Message)**。
    
- **典型协议：** HTTP (网页), FTP (文件), SMTP (邮件), DNS (域名解析)。
    

### ② 运输层 (Transport Layer)

- **任务：** 负责向两台主机中进程之间的通信提供通用的数据传输服务。
    
- **数据单位：** **报文段 (Segment)**。
    
- **核心协议：**
    
    - **TCP：** 面向连接的、可靠的传输。
        
    - **UDP：** 无连接的、尽最大努力交付的传输。
        

### ③ 网络层 (Network Layer)

- **任务：** 负责为分组交换网上的不同主机提供通信服务，选择合适的**路由**。
    
- **数据单位：** **IP数据报** 或 **分组 (Packet)**。
    
- **核心协议：** IP 协议、ICMP (控制报文协议)、ARP (地址解析)。
    

### ④ 数据链路层 (Data Link Layer)

- **任务：** 将网络层交下来的 IP 数据报组装成 **帧 (Frame)**，在相邻节点间的链路上传输。
    
- **功能：** 封装成帧、透明传输、差错检测。
    
- **典型设备：** 交换机、网桥。
    

### ⑤ 物理层 (Physical Layer)

- **任务：** 考虑如何在传输媒体上传输比特流，屏蔽掉具体传输媒体的差异。
    
- **数据单位：** **比特 (Bit)**。
    
- **典型设备：** 集线器 (Hub)、中继器。

## 3. 封装与拆封过程 (必记流程)

在发送端，数据每经过一层都会加上该层的**首部 (Header)**（链路层还会加上尾部）：

1. **应用层：** 产生数据报文。
    
2. **运输层：** 加上端口号等信息，封装成**段**。
    
3. **网络层：** 加上 IP 地址等信息，封装成**包**。
    
4. **数据链路层：** 加上 MAC 地址等信息，封装成**帧**。
    
5. **物理层：** 转化为 **0 和 1** 进行传输。

## 4. 期末考点模拟练习

### 概念题（填空/选择）

1. 在 OSI 参考模型中，实现数据加密和解密功能的层次是：__________。（答案：表示层）
    
2. TCP 协议属于体系结构中的第 ____ 层，而 IP 协议属于第 ____ 层。（答案：4/运输层，3/网络层）
    
3. 数据链路层的传输单位是 ______。（答案：帧）
    

### 计算题（综合应用）

> **题目：** 假设应用层数据长度为 $1000 \text{ B}$。在五层协议体系结构中，每一层增加的首部开销如下：
> 
> - 运输层：$20 \text{ B}$
>     
> - 网络层：$20 \text{ B}$
>     
> - 数据链路层：$18 \text{ B}$（包含首部和尾部）
>     
> - 物理层：不增加额外字节（简化情况）
>     
> 
> 问： > 1. 发送到物理链路上的总数据量是多少字节？
> 
> 2. 该数据在链路上的传输效率（有效载荷/总数据量）是多少？
> 
> **解答提示：**
> 
> 3. 总字节数 $L = 1000 + 20 + 20 + 18 = 1058 \text{ B}$。
>     
> 4. 传输效率 $\eta = \frac{1000}{1058} \approx 94.5\%$。
>
